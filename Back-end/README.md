## BackEnd

모방 대상 : 비즈메카

기능 : 로그인 기능, 메모, 근태관리, 쪽지, 일정

- 메모 : 분류 탭, 모든 메모, 날짜 , 휴지통

- 근태관리 : 당월 근태 출퇴근 시간, 휴가 신청 및 휴가 조회 기능

- 쪽지 : 같은 그룹내의 사람끼리 쪽지 보내기 기능, 보낸 쪽지 확인, 휴지통

- 일정 : 적어둔 메모 날짜별로 확인 및 휴가 체크
